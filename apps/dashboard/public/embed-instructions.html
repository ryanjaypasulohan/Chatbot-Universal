Installation Guide

Add your Wild Script chatbot to any website in under two minutes.

Step 1 — Copy the embed script

From the dashboard, create a chatbot and click Copy Script.

Step 2 — Paste before </body>

<!-- Wild Script Chatbot -->
<script>
window.AI_CHATBOT_WIDGET_CONFIG = {
  websiteId: 'YOUR_CHATBOT_ID',
  apiUrl: 'https://your-api.com/api/chat'
};
</script>
<script type="module" src="https://your-api.com/widget/embed.js"></script>

Platform examples

WordPress

Appearance → Theme File Editor → footer.php, or use a "Custom HTML" block in the footer widget area.

Shopify

Online Store → Themes → Edit code → theme.liquid, paste before </body>.

React / Next.js

useEffect(() => {
  const s = document.createElement('script');
  s.innerHTML = `window.AI_CHATBOT_WIDGET_CONFIG={...}`;
  document.body.appendChild(s);
  // load embed module
}, []);

Plain HTML

Paste the snippet at the end of your index.html before the closing body tag.

Troubleshooting

  • Ensure websiteId matches your chatbot in the dashboard.
  • API URL must point to your Wild Script API host (same origin in dev).
  • Crawl your site first so the bot has knowledge to answer from.
